# RefIn

RefIn is a job matching platform built with React, FastAPI, and PostgreSQL (SQLite for development) to support refugees, displaced individuals, and the unemployed in finding meaningful job opportunities during challenging times. Unlike general platforms, it is tailored specifically for this community, offering location and remote-based preferences while allowing employers to actively seek suitable candidates. Beyond job listings, RefIn fosters a supportive and inclusive environment that empowers its users.

The project's frontend code lives at: https://github.com/raphaeltannous/techtalks-refin-frontend
